# 机器学习/统计建模分析全流程 (Statistical Modeling & ML Analysis Workflow)

这个仓库包含了一个端到端的机器学习模型构建与统计建模分析流程。代码均使用 Jupyter Notebook (`.ipynb`) 编写，涵盖了从数据探索到模型可解释性的完整生命周期。

## 📂 项目文件及说明

本项目建议按照以下顺序执行 Notebook，以复现完整的分析流程：

| 文件名称 | 功能与内容描述 |
| :--- | :--- |
| **[`0.EDA.ipynb`] ./0.EDA.ipynb** | **探索性数据分析**：了解数据缺失值、异常值及变量间的关系。 |
| **[`1.标签构造.ipynb`] ./1.标签构造.ipynb** | **标签工程**：构建目标变量（Target Variable）。 |
| **[`2.数据清洗、标准化、划分.ipynb`] ./2.数据清洗、标准化、划分.ipynb** | **数据预处理**：缺失值处理、数据标准化、以及训练/测试集切分。 |
| **[`3.特征选择.ipynb`] ./3.特征选择.ipynb** | **特征选择**：筛选对预测最有贡献的变量，去除冗余特征。 |
| **[`4.各模型性能.ipynb`] ./4.各模型性能.ipynb** | **模型基线对比**：在未调参的情况下，评估多个基础模型的性能。 |
| **[`5.超参数调优.ipynb`] ./5.超参数调优.ipynb** | **超参数优化**：使用贝叶斯优化寻找最佳模型参数。 |
| **[`6.shap分析.ipynb`] ./6.shap分析.ipynb** | **模型可解释性**：利用 SHAP 值分析特征对预测结果的贡献度，理解黑盒模型。 |
| **[`7.学习曲线.ipynb`] ./7.学习曲线.ipynb** | **学习曲线诊断**：评估模型是否存在过拟合或欠拟合问题。 |
| **[`8.变量定义与基线特征表.ipynb`] ./8.变量定义与基线特征表.ipynb** | **变量字典与基线表**：记录所有参与建模的变量定义、分布情况以及基线特征表格。 |
| **[`9.超参数验证曲线.ipynb`] ./9.超参数验证曲线.ipynb** | **验证曲线分析**：可视化不同超参数对模型性能的影响，验证调优结果。 |
